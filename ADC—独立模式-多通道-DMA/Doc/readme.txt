【*】 程序简介 
-工程名称：ADC-多通道-DMA
-实验平台: 野火STM32 F407 开发板
-MDK版本：5.16
-ST固件库版本：1.8.0

【 ！】功能简介：
使用ADC多通道DMA模式测量电压

【 ！！】注意事项：

 跳线帽：使用跳线帽连接 板子右侧的VR<--->PB0排针，调节电位器可以改变电压
 跳线帽：使用跳线帽连接 板子右侧的LS<--->PB1排针，测量光照传感器电压
 
 外接电压范围为0~3.3V，超出范围可能导致烧坏
 
【 ！】实验操作：
电脑端使用串口调试助手，选择电脑与STM32相连的COM口，设置为115200-N-8-1，
复位开发板，即可接收STM32串口发送给电脑的数据。

调节电位器可以改变PB0的电压
改变光照环境可以改变PB1的电压

/*********************************************************************************************/

【*】 引脚分配
ADC：
	VR<--->PB0(通过跳线帽连接)
	LS<--->PB1(通过跳线帽连接)
	
	可外接0~3.3V<--->PA6
	
/*********************************************************************************************/

【*】 时钟

A.晶振：

-RTC晶振：32.768KHz

B.各总线运行时钟：
-系统时钟 = SYCCLK = AHB1 = 168MHz
-APB2 = 84MHz 
-APB1 = 42MHz

C.浮点运算单元：
  不使用
